# 🌸 Iris Flower Classification using Machine Learning

A modern **Machine Learning web application** built with **Streamlit** that predicts the species of an Iris flower based on its measurements using a **Random Forest Classifier**.

---

## 🚀 Live Demo

🔗 **Demo: https://irisflowerclassificationmlmodel.streamlit.app**

---

## 📖 About the Project

The Iris Flower Classification project predicts one of the three Iris flower species based on four input features:

* 🌿 Sepal Length
* 🌿 Sepal Width
* 🌸 Petal Length
* 🌸 Petal Width

The application features a modern dark-themed dashboard with an interactive interface for real-time predictions.

## 📊 Dataset

The project uses the **Iris Dataset**, one of the most popular datasets in machine learning.

**Target Classes:**

* Iris Setosa
* Iris Versicolor
* Iris Virginica

**Features:**

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## 🤖 Machine Learning Model

**Algorithm:** Random Forest Classifier

### Workflow

1. Load Iris Dataset
2. Train Random Forest Model
3. Accept User Inputs
4. Predict Flower Species
5. Display Prediction Confidence

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Iris-Flower-Classification.git

```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

